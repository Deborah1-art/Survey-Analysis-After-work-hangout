# Survey-Analysis-After-work-hangout
A Power BI survey analysis for planning an after-work hangout aimed at young professionals, focusing on preferred timing, audience demographics, and discussion topics.

This project presents an analysis of survey responses collected to plan a relaxed, after-work hangout for young professionals.

The goal of the event is to create a space for open conversations, controversial discussions, and unwinding after work hours.

Using Power BI, the data was analysed to understand audience preferences, ideal timing, and interest levels to ensure maximum participation and engagement.

## Objectives
Identify the most suitable day and time for the hangout
Understand the profile of the target audience
Measure interest and willingness to attend
Identify discussion topics participants are interested in
Support planning of an engaging and relevant event
